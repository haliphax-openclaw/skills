# Custom OpenClaw skills

These are custom OpenClaw skills I have developed in coordination with OpenClaw agents.

## Skills

- `deluge` - Manages torrents on a remote Deluge WebUI instance via JSON-RPC.
- `fan-out` - Fan out a list of tasks to subagents with unified tracking and status updates.
- `rtsp-snapshot` - Captures a still frame from an RTSP camera stream using ffmpeg.
